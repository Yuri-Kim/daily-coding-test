### 코딩테스트 연습 > 연습문제 > 같은 숫자는 싫어    
[문제](https://programmers.co.kr/learn/courses/30/lessons/12906)  
- 0부터 9까지의 숫자로 이루어진 배열에서 연속적으로 나타나는 숫자를 하나로 남기고 제거(순서유지)    

예시   

| arr | result |   
| --- | --- |    
| [1,1,3,3,0,1,1] | [1,3,0,1] |  
| [4,4,4,3,3] | [4,3] |  

제한 사항  
- 배열 arr 크기 : 1,000,000 이하의 자연수  
- 원소의 크기 : 0~9
