### 코딩테스트 연습 > 연습문제 > 서울에서 김서방 찾기   
[문제](https://programmers.co.kr/learn/courses/30/lessons/12919)  
- 배열 Seoul에서 "Kim"의 index 찾기  

예시   

| seoul | return |    
| --- | --- |      
| ['Jane', 'Kim'] | '김서방은 1에 있다' |  


제한 사항  
- seoul은 길이 1 이상, 1000 이하인 배열  
- seoul의 원소는 길이 1 이상, 20 이하인 문자열  
- Kim은 반드시 seoul 안에 포함
