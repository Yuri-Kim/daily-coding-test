### 코딩테스트 연습 > 연습문제 > 문자열 내 p와 y의 개   
[문제](https://programmers.co.kr/learn/courses/30/lessons/12915)  
- 문자열 내 P개수 y개수 비교 같으면 true, 다르면 false 리턴(대소문자 구분 안함)  

예시   

| s | answer |    
| --- | --- |      
| "pPooyY" | true |  
| "Pyy" | false |  


제한 사항  
- 문자열 s의 길이 : 50 이하의 자연수  
- 문자열 s는 알파벳으로만 이루어짐
